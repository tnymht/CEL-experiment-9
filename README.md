# Numerical Methods Lab – Differentiation & Integration

This repository contains MATLAB implementations of numerical methods used to approximate derivatives and definite integrals. The main objective of this project is to understand how these techniques are applied when analytical solutions are difficult and to compare their accuracy with exact results.

The work focuses on two key areas: numerical differentiation and numerical integration. In differentiation, the forward difference method is used after converting uneven data into equal intervals. The results show that the approximation error is higher at the starting point and improves for later values due to the nature of the method.

For integration, the given function is evaluated using the trapezoidal rule and Simpson’s rules. The trapezoidal rule shows larger error when fewer intervals are used, but its accuracy improves as the number of intervals increases. Simpson’s methods provide better results as they approximate the curve more accurately, with Simpson’s rules giving values closer to the analytical solution.

Overall, this project highlights that numerical methods are effective tools for solving practical problems, but their accuracy depends on the method and step size. It also demonstrates that higher-order methods and increased intervals lead to better approximations, even though some error is always present.
